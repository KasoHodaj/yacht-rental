# Alex Yacht Website

Small one-page website I built for a yacht rental client in Paxos, Greece.

## What it has

- Hero section
- About section + captain card
- Gallery (with extra images on "View Full Gallery")
- Booking buttons (WhatsApp, Viber, Instagram, Call)
- Footer with newsletter placeholder

## Built with

- HTML
- CSS
- JavaScript (vanilla)

## Run

Just open `index.html` in your browser (or use Live Server in VS Code).

## Quick edits

- Text/content: `index.html`
- Styling/colors/buttons: `css/style.css`
- Interactions: `js/script.js`
